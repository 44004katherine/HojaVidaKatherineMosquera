# Carpeta de Imágenes de Perfil

Coloca aquí tu foto de perfil con el nombre:
`profile.jpg` o `profile.png`

Actualmente el portafolio está configurado para leer esta imagen de forma local desde esta ruta:
`/images/profile.jpg` (que corresponde a este directorio en el servidor).

Puedes reemplazar esta imagen subiendo tu propia foto en formato JPG y renombrándola a `profile.jpg` dentro de esta carpeta `/public/images/`.
